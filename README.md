# CAMPUS ONE School - Official Website



A **modern, elegant, and fully functional** multipage school website built for **CAMPUS ONE School**, Nellore, Andhra Pradesh.
and this website delivers a trustworthy, professional, and user-friendly digital experience.

### ✨ Key Features

- Fully responsive design (Mobile, Tablet & Desktop)
- Beautiful hero section with Admissions 2026-27 CTA
- Dark/Light mode toggle with localStorage persistence
- Interactive mobile hamburger menu
- Working Admission & Contact forms with real-time validation
- Role-based fake login system (Admin, Teacher, Student/Parent)
- Real Google Maps integration
- Notice ticker, smooth scrolling & back-to-top button
- Rich sections: Academics, Facilities, Achievements, Gallery, News & Events

### 🎨 Design Philosophy

This project follows the visual language of Indian government educational institutions with a refined and premium touch. It blends the trust and discipline of Kendriya Vidyalaya Sangathan (KVS) style with modern web design principles.

**Official Color Palette:**
- **Primary**: Orange `#ff9f08`
- **Secondary**: Navy Blue `#003087`
- **Accent**: Gold `#f4c430`

The main goal was to create a **trustworthy, professional, and user-friendly** digital presence for CAMPUS ONE School that reflects excellence in education.
